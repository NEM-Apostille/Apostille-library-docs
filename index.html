Apostille Library

A novel & holistic blockchain notarization and timestamping with transferable, updatable, branded, and conjointly owned notarizations.

Get Started →

Ownership

Digital & real world notorized assets can now be owned or conjointly owned on the blockchain.

Transferable

Notarization ownership can be transfered from party to party on the blockchain.

Updatable

Notorized assets can receive updates over time.

Brandable & auditable

Control over notarizing parties with full transparency on the blockchain.

# Use It

# install
npm install apostille-library # OR yarn add apostille-library

COMPATIBILITY NOTE

Apostille-library works only with catapult (NIS2).
